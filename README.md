## 📂 Repository Structure

Place all your work inside the `tasks` folder. Create a dedicated subfolder named after your **Your Name**.

```text
repository-root/
│
├── tasks/
│   ├── *your-name*/
│   │   └── [Add your task files/folders here] 👈
│   │
│   └── .gitkeep
│
└── README.md
```

## 🛠️ Step-by-Step Submission Guide

### 1️⃣ Clone the Repository
copy paste the repo url from code dropdown:
```bash
git clone https://github.com/ieee-vssut-sb/cs_society_tasks.git
cd this-repo-name
```

### 2️⃣ Create a New Branch (Recommended)
```bash
git checkout -b feature/task-1
```

### 3️⃣ Add Your Files
1. Navigate into the `tasks/` directory.
2. Create your personal subfolder (e.g., `tasks/aswin/`).
3. Add your solution files inside your newly created folder.

### 4️⃣ Commit & Push Your Changes
```bash
# Stage your specific folder
git add tasks/your-github-username/

# Commit with a meaningful message
git commit -m "add task for your-name"

# Push the branch to the remote repository
git push origin feature/task-1
```

### 5️⃣ Open a Pull Request (PR)
1. Go to the repository page on GitHub.
2. Click on the **Compare & pull request** button next to your recently pushed branch.
3. Title your PR clearly (e.g., `Submission: Your-Name`).
4. Submit the Pull Request for review! 🎉

---

## ⚠️ Important Instructions

> **Double-check your directory structure before pushing.** Ensure all your files are inside your personal subdirectory(your name) under the `tasks/` folder. Do not push files directly to the root directory or into another contributor's folder. 
>
> **Your submitted task folder name should be your name.**
>
> **Submit your pull request before the specified deadline.**
>
Good luck with your first task! If you run into any issues or have questions, feel free to reach out. Happy coding! 💻✨
